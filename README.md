# LittleWins

## Overview
We decided to create an online task manager embedded with supportive rhetoric to encourage users to complete their self-assigned tasks. This includes features such as logging in, creating, updating, and deleting tasks and task groups, assigning tasks to groups, and sorting tasks matching specified criteria. Additionally, once a user has created enough tasks in a given month, the "wrapped" feature unlocks which analyzes user habits provides suggestions to motivate users to keep at their tasks or suggest ways to improve. 

## Course Information

## Functional Features
Users can:
- Create accounts and update passwords
- Create tasks with due dates, effort estimation, and priority fields
- Sort tasks based on due date, estimated effort, and priority
- Update tasks due dates, effort, or priority
- Update the status of a task between "not started", "in progress", and "complete"
- Delete any unneeded tasks
- Create task groups and add tasks to those custom groups
- Color-code task groups
- Modify or delete groups with dynamic task reassignment
- View an overview of monthly progress

## Non-Functional Features
- Passwords and security question answers are hashed and salted ensuring secure management of sensitive data
- Session information can be saved reducing the amount of times a user must log in
- Database updates happen in real-time, preventing delay in use
- The system supports all modern browsers and has accessible UI for all common smart devices
